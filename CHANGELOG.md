# Changelog

All notable changes to `mutabakat` will be documented in this file.

## v2.0.0 - 2025-12-17

### Changed
- Upgraded to FilamentPHP 4
- Updated minimum PHP version to 8.3
- Updated all dependencies for Filament 4 compatibility

## v1.0.0 - 2024-11-15

### Added
- Initial release
- FilamentPHP 3 plugin for mutabakat (reconciliation) management
- Support for Laravel 11 and 12
- Complete CRUD operations
- Soft deletes support
- Comprehensive table structure with all required fields
- Configurable status options
- Multi-currency support
- Installation command
- Resource pages for list, create, and edit operations
- Filters for status and soft-deleted records
- Bulk actions support
